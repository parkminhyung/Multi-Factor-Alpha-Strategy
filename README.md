# Multi-Market Multi-Factor Alpha Strategy - Backtest Report

> **A systematic equity selection pipeline applied independently and without structural modification across three major Asian markets: KRX (Korea), JPX (Japan), and CHN (China A-shares)**

![Performance Chart](multi_market_performance.png)

---

## Overview

This report documents a systematic quantitative strategy originally developed for the Korean Stock Exchange (KRX) and subsequently deployed to the Tokyo Stock Exchange (JPX) and China A-share market (CHN) - using an identical pipeline in each case. The strategy combines a three-stream multi-factor signal generation framework with a two-stage machine learning ensemble, applied to large-cap equities in each respective market.

Each backtest spans approximately 9-10 years, covering multiple distinct market regimes:

- **2018** - US-China trade-war selloff
- **2020** - COVID-19 crash and recovery
- **2021** - Global growth rally and China tech crackdown
- **2022** - Aggressive rate-hike bear market
- **2023** - China post-reopening stimulus; Nikkei secular breakout
- **2024-2025** - JPY depreciation cycle; Korea exceptional bull run

All results are **fully out-of-sample**. Unlike a conventional train/test split, the rolling walk-forward structure means that every single prediction across all three markets - from inception to the final date - was generated by a model that had never seen the data it was predicting. Each fold trains exclusively on prior history and predicts only the immediately following period. The full equity curve of each market is therefore a sequential accumulation of OOS predictions, not a mix of in-sample and out-of-sample results.

---

## Strategy Framework

The pipeline operates as a sequential four-stage process, **identical across all three markets**:

```
SIGNAL PIPELINE
UNIVERSE -> FACTORS -> SELECTION -> PORTFOLIO

Universe: market-specific large-cap universe
Factors: handcrafted + template-based + GP-evolved
Selection: two-stage ML ensemble (candidate filter -> rank/score)
Portfolio: volatility-adjusted equal weighting, HMA exit + stop-loss, T+1 settlement
```

**Factor streams:**

Factors are sourced from three complementary streams. Handcrafted conventional factors encode well-established financial intuitions (value, momentum, quality, liquidity). Template-based expression factors systematically combine price and volume primitives across multiple time horizons. GP-evolved factors are generated by a genetic programming engine that evolves novel expressions optimised directly against return predictability - re-evolved at each quarterly fold boundary using only that fold's training window.

All three streams pass through an IC/ICIR filter computed **within each walk-forward fold**, using only the fold's training period. No future return data participates in factor selection at any stage. Specific factor definitions, GP parameters, and model architectures are not publicly disclosed.

**Walk-forward structure:**

At each rebalance, the pipeline trains on a rolling window of prior data, validates on the subsequent period, and predicts only the immediately following test window. This process repeats sequentially across the full backtest horizon, producing a chain of OOS predictions with no look-ahead leakage at any fold boundary.

### The Asymmetric Payoff Structure

All three markets share the same defining characteristic: a **below-50% win rate paired with a win/loss ratio well above 2**. This is not a coincidence - it is a structural consequence of the exit design. The HMA-based exit holds winning positions through the trend and cuts losing positions quickly; the stop-loss enforces a hard floor on downside per trade. The result is a systematic truncation of the left tail and extension of the right tail of the per-trade return distribution.

```
KRX :  Win 43.0% of trades  *  avg +9.51%  vs  lose 57.0%  *  avg -3.45%
        Expected value per trade = (0.43 * 9.51%) + (0.57 * -3.45%) = +2.12%

JPX :  Win 48.4% of trades  *  avg +5.98%  vs  lose 51.6%  *  avg -2.60%
        Expected value per trade = (0.484 * 5.98%) + (0.516 * -2.60%) = +1.55%

CHN :  Win 46.2% of trades  *  avg +8.12%  vs  lose 53.8%  *  avg -2.85%
        Expected value per trade = (0.462 * 8.12%) + (0.538 * -2.85%) = +2.22%
```

A consistently positive expected value per trade, confirmed across 5,828 combined trades, is the most direct evidence of a structural edge at the individual signal level.

### Exit Mechanism Breakdown

Over 96-97% of exits across all three markets are HMA-triggered (trend reversals), with stop-loss exits comprising only 2-2.3%. This distribution reflects the strategy's primary reliance on trend exhaustion as the exit signal rather than loss-cutting - stop-losses serve as a hard backstop for gap-down events rather than the primary exit mechanism.

The average stop-loss return in CHN (-8.66%) is noticeably deeper than the nominal -7% trigger. This is a direct consequence of China A-share price limits: when a stop-loss is triggered at a daily limit-down, the exit is deferred to the next available session. The code implements this deferral explicitly (deferred_exits logic), but the resulting slippage cannot be eliminated.

## Limitations

Transparency requires an honest disclosure of the following constraints. All limitations apply to all three markets unless explicitly noted.

**1. Universe Survivorship Bias - most significant caveat (all markets)**

Investment universes are constructed using market-capitalisation rankings near the data end date and applied retroactively to 2016. Stocks that grew into the top universe by the end of the sample period are included from inception; stocks that declined, were acquired, or were delisted are excluded. This introduces a form of survivorship bias: the backtest universe skews toward companies that performed well over the sample period.

Point-in-time (PIT) universe reconstruction - using the actual constituents of each index at each historical date - would be required for a fully unbiased evaluation. This is the single most important caveat in this report, and any live deployment must use a PIT-consistent universe.

**2. Execution Assumptions (all markets)**

Entry prices are modelled at the closing auction price. While operationally achievable via exchange closing auction mechanisms (KRX closing auction, TSE closing cross, Shanghai/Shenzhen call auction), actual fill prices may deviate marginally from the published closing price in practice. Market impact from large orders is not modelled; the current backtest is valid under the assumption that position sizes are small relative to average daily volume.

**3. Regime Sensitivity - Structural Low-Beta Drag (all markets)**

The strategy's low-beta profile (beta ~ 0.26-0.37) generates returns primarily through stock selection rather than broad market exposure. This is an advantage in flat, declining, or stock-picker's markets, but creates a structural headwind in sustained, broad-based momentum rallies where index returns are driven by indiscriminate multiple expansion. KRX 2025 (raw alpha -40.9%, Jensen's alpha +6.7%) and JPX 2023 (raw alpha -4.9%) are the clearest illustrations of this trade-off during the sample period.

**4. Serial Autocorrelation (all markets)**

Daily returns exhibit positive lag-1 autocorrelation (rho ~ +0.08 to +0.13), which inflates the apparent precision of naive OLS t-statistics. This is addressed by the Newey-West HAC correction reported above, but analysts relying on standalone OLS t-statistics should apply appropriate corrections.

**5. Partial CSI300 Benchmark Data (CHN only)**

Complete CSI300 daily rebased data was not available for the full CHN backtest period. As a result, full CAPM decomposition and benchmark-relative year-by-year comparison for CHN are computed over the available overlap period only. Annual Sharpe ratios are presented as a partial substitute.

**6. Price-Limit Slippage (CHN only)**

China A-shares are subject to daily price limits (±10% main board, ±20% ChiNext and STAR Market). When a stop-loss is triggered on a day that closes at the limit-down price, the exit order cannot be filled and is deferred to the next available session. The code implements explicit deferral logic, but the resulting slippage is unavoidable - reflected in the average realised stop-loss return (-8.66%) running deeper than the nominal -7% trigger.

**7. Currency Denomination (JPX only)**

JPX results are denominated in Japanese Yen (JPY). The 2024-2025 period saw significant JPY depreciation. Investors computing returns in USD, KRW, or other currencies would experience materially different outcomes depending on their hedging policy and the associated cost of carry.
